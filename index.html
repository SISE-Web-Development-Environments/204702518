3 Differences between Web1, Web2 and Web3:

1. Web 1.0 is Mostly Read-Only while Web 2.0 is Read-Write and Web 3.0 is Portable and Personal.
2. Web 1.0 is about Home Pages while Web 2.0 is Blogs or Wikis and Web 3.0 is about Live Streams.
3. Web 1.0 works with Directories while Web 2.0 works with Tagging and Web 3.0 is about User Behaviour and User Learning.

Use Case : Performing a Query Search in a Search Engine about Medicine side effects:

1. In Web 1.0 a query search in a Search Egine will return macro content, for example it will return
information about what are side effects and a definition for it.
2. In Web 2.0 a query search in a Search Engine will return micro content with tags, for example it will return
side effects for specific and well known and tagged medicines.
3. In Web 3.0 a query search in a Search Engine will return automated tagged micro content.
